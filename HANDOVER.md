| 자사홈 | 플랫폼 1위 예약수 | 자사 직접 예약수 | 자사 전환율 (자동계산) |

#### 콘텐츠 대시보드 전면 재설계
- **탭 순서 변경**: OSMU → **자사전환율(2번)** → 공지동기화 → 출장플랜
- **OSMU 1탭**: 블로그 수동 카운팅 삭제 → 구글시트 콘텐츠스케줄 기반 채널별 월간 발행 자동 집계
- **OSMU 흐름도**: 유튜브→인스타→블로그→홈페이지 메인 흐름 / 카카오·쓰레드 소통·공지 채널로 분리, 각 채널 역할 명시
- **자사전환율**: 플랫폼1위 예약·자사 예약·전환율★·전월대비 4카드 + 목표달성률 바 + 월별 추이 차트
- **공지동기화**: 테이블 → 타임라인 UI, 최신 공지 먼저 표시, 채널 카드 클릭 → 해당 채널 공지 페이지 바로가기
- **출장플랜**: Google Sheets 연동 추가 → 크로스 디바이스 저장 보장

#### 크로스 디바이스 완전 보장
- `saveFaq()` Google Sheets 동시 저장 추가 (기존 localStorage만 저장하던 버그 수정)
- `saveAll()` URL 전체 데이터 Google Sheets payload 포함
- 모든 데이터 저장 흐름: **localStorage(캐시) + Google Sheets(원본)** 이중 구조

#### Apps Script 안정화
- `readSyncLog()`: 날짜 Date 객체 → `yyyy-MM-dd` 포맷 변환 추가
- `readOsmu()`: month 필드 Date 객체 처리 + 전체 필드 빈값 안전처리
- `writeOsmu()` / `writeSyncLog()`: undefined 필드 빈문자열 처리
- `readTrip()` / `writeTrip()` 함수 신규 추가
- 출장플랜 시트 (`출장플랜`) 자동 생성 추가
